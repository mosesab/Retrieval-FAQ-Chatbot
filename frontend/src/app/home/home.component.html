Chatbot Features

Question Segmentation

The Bot can process multiple questions at once, it can provide an answer to questions. It can also recommend similar answers to other questions.

Vector Space Model

A Vector Space Model is used by a computer to learn and remember words as numbers or vectors. The bot uses vectors to model words as numbers.

Instant Answers to Questions

The bot uses APIs to interact with the backend so as to answer questions very quickly. Tests have been run to ensure the APIs are safe, secure and most importantly performant.
